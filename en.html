Convert text to vector

On Windows

Open the "Notepad" application and paste the copied data.

Select "File" and "Save As" and save the file as "(name of your choice).svg" to a location of your choice. Load this data into Scratch to use it. Paint it with colors as you like.

On Mac

From the Dock at the bottom of the screen, open "Launchpad" with the rocket symbol and select the "TextEdit" application.

Press "New Document" in the lower left corner to create a text file.

From the bar at the top of the screen, select "Format" -> "Make as standard text", then paste the copied data and save it anywhere you like with any name you like.

Open Finder, go to "Finder" at the top of the screen, open "Preferences" and turn on "Show all file extensions".

Select the saved file and press the Return key to rename the file to "(name of your choice).svg". Load this data into Scratch to use it. You can color it if you like.
